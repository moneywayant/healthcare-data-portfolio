### Diabetes Indicators & Physical Activity

**Summary:**  
Using BRFSS 2015 data, this project explores how diabetes correlates with lifestyle factors like physical activity, BMI, and smoking. SQL was used for segmentation, and Power BI provided visual context.

**Tools Used:** SQL, Power BI  
**Skills Highlighted:** Health behavior analytics, DAX, interactivity

**Key Features:**  
- Diabetes rates by gender, activity level, smoking  
- Average BMI by diabetes status  
- Interactive dashboard with age group filters
