### Medical Claims Denial Analysis

**Summary:**  
This project investigates patterns in denied medical claims. I used Excel for preliminary analysis and Power BI to create a dynamic dashboard that breaks down denial trends by payer, procedure, and denial reason.

**Tools Used:** Excel, Power BI  
**Skills Highlighted:** Claims analysis, visual storytelling, trend detection

**Key Features:**  
- Denied amount KPIs  
- Denial rate by payer and procedure code  
- Visual breakdown by denial reason and insurance type
