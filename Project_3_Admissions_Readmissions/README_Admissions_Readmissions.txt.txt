### Patient Admissions & Readmissions SQL Insights

**Summary:**  
This project uses SQL to explore trends in hospital admissions, focusing on billing, admission type, and diagnosis. The goal was to reveal high-cost conditions and demographic trends in admissions.

**Tools Used:** SQL (BigQuery), Power BI  
**Skills Highlighted:** SQL querying, aggregation, CASE logic, joins

**Key Queries:**  
- Most common conditions  
- Avg. billing by gender and admission type  
- Readmission trends and insurance provider analysis
